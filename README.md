# Ajax Chat

A [Flarum](http://flarum.org) extension. no desc.

## Install

Install with composer:

```sh
composer require nearata/flarum-ext-ajax-chat:"*"
```

## Update

```sh
composer update nearata/flarum-ext-ajax-chat:"*"
php flarum migrate
php flarum cache:clear
```
